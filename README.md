Repository ini berisi proses eksplorasi dan eksperimen machine learning untuk kasus **Fraud Detection** menggunakan dataset transaksi perbankan.
Eksperimen mencakup tahap preprocessing data, pembagian data latih dan uji, serta evaluasi performa model klasifikasi.
Seluruh proses eksperimen dilakukan menggunakan Jupyter Notebook.
Folder `preprocessing` berisi script dan hasil preprocessing data sebelum pemodelan.
Repository ini dibuat sebagai bagian dari submission **Proyek Akhir Membangun Sistem Machine Learning (Dicoding)**.
